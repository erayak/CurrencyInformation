# CurrencyInformation ( Node Js )

***

### Installation
```sh
$ npm install -g
$ currency
```

| Linux Users `sudo npm install -g ` after `currency`

### Alternative
```sh
$ npm install
$ node app.js 
```

## USE
1. Type 1 for currency information.
2. Type 2 to calculate the currency.
3. Click the Enter key.
